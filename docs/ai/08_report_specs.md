## 1. 青色申告決算書 (個人事業主) — 令和7年版

|----|------|------|
| 1  | 売上（収入）金額 | 雑収入を含む |
| 2  | 売上原価 | 期首棚卸高 + 仕入金額 - 期末棚卸高 |
|    | 　期首棚卸高 | |
|    | 　仕入金額 | |
|    | 　小計 | |
|    | 　期末棚卸高 | |
| 3  | 差引金額 (売上総利益) | |
| 4〜| 経費の部 | 給料賃金/外注工賃/減価償却費/貸倒金/地代家賃/利子割引料/租税公課/その他の経費 |
|    | 経費合計 | |
| 5  | 各種引当金・準備金等 | |
| 6  | 差引所得金額 (営業利益相当) | 売上総利益 - 経費合計 |

- 旅費交通費
- 通信費
- 広告宣伝費
- 接待交際費
- 損害保険料
- 修繕費
- 消耗品費
- 福利厚生費
- 雑費
- その他の経費

### 1-2. 月別売上仕入金額 (第1表 付記)

| 月 | 売上(収入)金額 | 仕入金額 |
|----|--------------|---------|
| 1月〜12月 | 各月合計 | 各月合計 |
| 合計 | 年間合計 | 年間合計 |

### 1-3. 貸借対照表 (第2表)

**資産の部:**
- 現金・預金
- 売掛金
- 棚卸資産
- 前払費用
- その他の流動資産
- 固定資産 (有形・無形)
- 一括償却資産
- 繰延資産
- 資産合計

**負債の部:**
- 買掛金
- 未払金
- 借入金
- その他の負債
- 負債合計

**元入金等:**
- 元入金
- 事業主貸
- 事業主借
- 当期純利益 (または損失)
- 期末元入金

### 1-4. 源泉徴収税額

- 本年中の所得税額の源泉徴収税額: 金額

## 2. 法人決算書 (中小法人向け)

### 2-1. 貸借対照表 (様式第5号準拠)

**資産の部 — 流動資産:**
- 現金及び預金
- 受取手形
- 売掛金
- 有価証券
- 棚卸資産
- 短期貸付金
- 前払費用
- その他
- 貸倒引当金 (△)
- 流動資産合計

**資産の部 — 固定資産:**
- 有形固定資産 (建物/機械装置/工具器具備品/土地/建設仮勘定)
- 無形固定資産 (のれん/特許権/ソフトウェア)
- 投資その他の資産 (投資有価証券/長期貸付金/繰延税金資産/その他)
- 固定資産合計

**資産の部 — 繰延資産:**
- 繰延資産合計

**資産合計**

**負債の部 — 流動負債:**
- 支払手形
- 買掛金
- 短期借入金
- 1年内返済長期借入金
- 未払費用
- 未払法人税等
- 前受収益
- その他
- 流動負債合計

**負債の部 — 固定負債:**
- 長期借入金
- 退職給付引当金
- その他
- 固定負債合計

**負債合計**

**純資産の部:**
- 資本金
- 資本準備金
- その他資本剰余金
- 利益準備金
- その他利益剰余金 (繰越利益剰余金)
- 自己株式 (△)
- 純資産合計

**負債純資産合計**

| 行 | 科目 |
|----|------|

### 2-3. 株主資本等変動計算書

列: 資本金 / 資本剰余金 / 利益剰余金 / 自己株式 / 純資産合計

行:
- 当期首残高
- 当期変動額 (配当/当期純利益/自己株式取得・処分/その他)
- 当期変動額合計
- 当期末残高

- 重要な会計方針
- 資産の評価基準
- 固定資産の減価償却方法
- 引当金の計上基準

## 4. 実装方針

- HTML テンプレート: `templates/reports/` 配下
- CSS: A4 印刷向け (`@page { size: A4; }`, `page-break-*`)
- 金額表示: 千円未満切り捨て or 円単位 (青色申告は円単位)
- 金額フォーマット: `number_format()` で3桁カンマ
- 赤字(マイナス)は △ 表示 または マイナス符号
- PHPテンプレートで `<?= htmlspecialchars($var) ?>` を使用
